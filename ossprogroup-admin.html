CMS léger — modification rapide

Ce panneau modifie directement les ids existants dans ossprogroup.html, puis peut exporter un fichier HTML final avec vos images/textes.
Aucun upload lourd : sélectionnez les images dans ossprogroup/MEDIA.

Hero — textes

IDs : hero-tag, hero-l1, hero-em, hero-l3, hero-sub

À propos — stats + texte

IDs : stat-1..4, stat-1-lbl..4-lbl, about-p1, about-p2

Réalisations — 3 projets

IDs : proj-*-cat, proj-*-title, proj-*-year

Médias — images site

Hero slider: slide-0..2 (background-image) • About: about-photo (src) • Portfolio bg: proj-1-bg..3-bg (background-image) • Logo: logo-nav/logo-footer (src)
Les images viennent de ossprogroup/MEDIA.
Si l’image ne s’affiche pas : vérifiez le nom exact du fichier dans Média.
